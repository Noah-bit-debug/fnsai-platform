Privacy

The extension never reads page content, form values, or passwords. These settings control what metadata is recorded.

Allow page title tracking
Send page titles (e.g. "Q4 Report — Google Sheets") to the server for richer reports. Disabled by default.
Show domain in reports

What is never collected: form contents, passwords, clipboard data, page body text, file contents, or any personally identifiable information beyond what you explicitly configure above.
